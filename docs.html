Introduction

You've made it to easy-contribute so I'm assuming you have learned the basics of HTML and CSS, maybe even a little JavaScript. You have started learning the basics of git hub but you need some real scenario to put your skills to action in order to fully understand it.

In this documentation you will walk through the steps to contribute to an open source project.

Copying the Repository

The first thing you need to do is to get a copy of the source code off of GitHub and on to you computer.

Let's go to the GitHub repository for easy-contribute

From there you want to click the Code button and copy the clone url which should look something like this https://github.com/chemond/easy-contribute.git you can go to your terminal and enter in the following command

git clone https://github.com/chemond/easy-contribute.git this is code
